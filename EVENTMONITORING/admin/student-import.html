Template & Format Guide

Required Columns

  • Student ID — Format: K-0001, 1-0001, 10-0001 (Grade-Serial)
  • First Name — Student's first name
  • Last Name — Student's last name

Optional Columns

  • Middle Name — May be left blank
  • Suffix — e.g. Jr., Sr., III — May be left blank
  • Birth Date — YYYY-MM-DD (if available)
  • Gender — male, female, or other
  • Email — Optional, only used for QR notification

Guardian Columns (Required)

  • Guardian1_Phone … Guardian1_Address — At least 1 guardian is required
  • Guardian2_Phone … Guardian2_Address — Optional second guardian
  • Relationship values: mother, father, legal_guardian, other
  • If phone number already exists, the existing guardian is linked instead of duplicated

Notes

  • Section is taken from Step 1 selection for all imported rows
  • Student IDs that already exist in the database are skipped
  • Do not delete the header row
  • Remove sample data before importing
